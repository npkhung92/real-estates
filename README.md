# real-estates
A simple real estate listing app
============================================

## Short brief:

A simple demo application showing a list of real estates

## Architecture:

- Layered Clean Architecture with multiple modules
- Kotlin Coroutines with Flow for asynchronous execution
- Room Database for local persistence
- Retrofit for remote network calls
- Hilt for dependency injection
- Jetpack Compose for UI implementation
- Jetpack Navigation
- Unit tests covered using Mockk & Turbine (testing flow)
